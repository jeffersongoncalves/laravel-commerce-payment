# Payment

[![Latest Version on Packagist](https://img.shields.io/packagist/v/jeffersongoncalves/laravel-commerce-payment.svg?style=flat-square)](https://packagist.org/packages/jeffersongoncalves/laravel-commerce-payment) [![Total Downloads](https://img.shields.io/packagist/dt/jeffersongoncalves/laravel-commerce-payment.svg?style=flat-square)](https://packagist.org/packages/jeffersongoncalves/laravel-commerce-payment) [![License](https://img.shields.io/packagist/l/jeffersongoncalves/laravel-commerce-payment.svg?style=flat-square)](LICENSE.md)

Commerce payment module — payment collections, payments, sessions, captures, and refunds.

## Installation

```bash
composer require jeffersongoncalves/laravel-commerce-payment
```

## Usage

The service provider is auto-discovered. Publish migrations and use the module models and services directly. See the [umbrella package](https://github.com/jeffersongoncalves/laravel-commerce) for the full ecosystem.
